# appkit-mcpapp

A FastMCP 3.0-based MCP server component for the appkit framework.

## Features

- FastMCP 3.0 server integration
- Built on appkit-commons and appkit-mantine
- Type-safe MCP server development

## Installation

```bash
pip install appkit-mcpapp
```
